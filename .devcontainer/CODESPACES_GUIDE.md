# 🚀 HEYTEA DIY GitHub Codespaces 使用指南

## 🎯 快速开始

1. **等待初始化** - Codespace 启动后会自动安装依赖并构建项目（约 2-3 分钟）
2. **访问界面** - 初始化完成后会自动启动 UI，在"端口"面板找到端口 5969 并打开转发地址
3. **开始使用**


```

## 💡 常用命令

```bash
# 启动 UI 界面
npm start


```

## 🔧 端口说明

- **5969** - Web UI 界面 + API 服务端口

## 🚨 常见问题

**Q: 服务没有启动？**
A: 在终端运行 `npm start` 重新启动

**Q: 依赖安装失败？**
A: 运行 `npm install` 重新安装依赖
